STEP ONE
01
授权绑定店铺
抖音、抖店、淘宝、天猫等多平台授权管理

完整操作流程

进入系统 → 渠道营销 → 账号管理 → 选择平台(抖音/抖店/淘宝/天猫) → 点击"新增/更新授权" → 填写店铺信息 → 确认授权绑定

🔗 店铺授权管理界面

多平台授权管理

界面展示了渠道营销中的账号管理功能，包含抖音、抖店、淘宝光合等多个平台的授权配置选项，支持新增/更新授权操作，为用户提供统一的店铺平台管理入口

🎯 支持平台
  • 抖音账号视频发布授权
  • 抖店店铺商品管理授权
  • 淘宝店铺数据同步授权
  • 天猫店铺管理授权
⚡ 核心功能
  • 一键快速授权绑定
  • 授权状态实时监控
  • 多平台统一管理
  • 权限安全管理控制
💡 重要提示
确保各平台授权状态正常，建议定期检查和更新授权信息，避免因授权过期影响正常使用。
STEP TWO
02
建立商品联系
手动创建商品绑定 OR 授权后自动识别同步店铺商品信息

完整操作流程

选择管理方式 → 手动创建(商品管理→创建商品→填写详细信息→确认添加) OR 自动同步(渠道营销→店铺商品→系统自动识别→筛选查看) → 建立商品库

📝 方式A：手动创建商品绑定
手动创建商品界面

展示商品管理页面的创建商品弹窗，支持人工创建和Excel导入两种方式，允许用户手动输入商品标签、货号、商品名称等详细信息，提供新增一行功能便于批量录入

🎯 适用场景
  • 新商品精准录入管理
  • Excel表格批量导入
  • 个性化商品信息设置
  • 商品数据完全可控
🤖 方式B：自动识别同步商品
自动同步商品界面

展示渠道营销模块下的店铺商品管理功能，通过淘宝大猫和抖店等平台标识，自动同步已授权店铺的商品信息，显示商品图片、名称、价格等详细信息，支持批量管理和筛选

⚡ 适用场景
  • 已有店铺快速对接
  • 智能筛选查看
  • 商品信息同步管理
  • 实时更新商品状态
🎯 选择建议
手动创建：适合新商品录入，数据精确可控　｜　自动识别：适合已有店铺商品，快速同步管理
STEP THREE
03
直播间配置
直播标签属性手动个性配置(可自然语言自定义)

完整操作流程

标签配置 → 直播切片 → 直播账号管理 → 标签配置 → 选择已有标签/新建个性标签 → 可自然语言自定义 → 确定保存配置

🏷️ 智能标签配置系统

标签配置管理中心

展示直播账号管理下的标签配置功能，已选择的标签包含商品外观、细节讲解、适应场景等多个维度，同时提供所有标签分类管理，包括通用类目（男鞋、女鞋、童鞋、服饰等）以及详细的功能特性标签

🏷️ 标签分类
  • 智能切片专用标签
  • 商品属性分类标签
  • 自然语言自定义标签
  • 个性化场景标签
⚙️ 配置功能
  • 个性化标签创建
  • 自然语言自定义
  • 标签使用统计
  • 智能推荐标签
STEP FOUR
04
智能直播录制剪辑
手动推流操作(url链接/本地内容/obs)，智能成片制作，内容分类管理

完整操作流程

直播录制 → 创建单场直播 → 选择推流方式(url链接/本地内容/obs) → 选定需要剪辑店铺商品清单配置 → 智能剪辑 → AI智能开始直播间录制剪辑 → 生成智能成片，智能高光片段 → 制作分类管理 → 选择保存位置(素材空间/本地)

🎬 智能直播录制与剪辑系统

直播项目创建

支持多种推流方式，包括url链接、本地内容、obs推流等，灵活适应不同直播场景

直播前商品清单配置管理

直播状态管理的核心功能界面，用户可在直播开始前通过"商品清单"功能预先选定需要生成直播切片的目标商品。系统会记录这些商品信息，在直播过程中AI智能识别技术会自动监测视频中出现的商品场景，精准匹配预设的商品清单，实现商品与视频内容的智能关联，为后续的自动剪辑和成片生成提供准确的数据支撑

AI智能内容分析

AI自动识别直播内容，智能生成商品推荐，自动标签分类，提升内容管理效率

智能商品匹配与剪辑工作台

展示直播剪辑的核心工作界面，左侧为直播时间轴和内容片段管理，右侧显示商品匹配区域。用户可在直播前通过商品清单选定需要生成切片的商品，系统AI会智能识别视频中的商品场景，自动匹配对应商品信息，实现精准的商品-视频关联，为后续的智能剪辑提供数据基础

智能成片列表管理

展示AI智能生成的视频成片列表，包含视频预览图、时长信息、标题描述，显示智能生成的多维度标签（特别好穿、适应场景、百搭、舒适等），提供关联商品信息和批量导出功能，支持列表和图标两种查看模式

📹 推流方式
  • url链接直播转录
  • 本地内容上传
  • obs推流实时录制
  • 商品清单配置
🤖 AI智能剪辑
  • 智能成片自动生成
  • 智能高光片段识别
  • 内容分类管理
  • 多格式视频输出
💾 存储管理
  • 素材空间云端存储
  • 本地文件导出
  • 分类标签管理
  • 内容版本管理
STEP FIVE
05
关联商品同步发布
确定视频内容关联商品，同步发布指定媒体

完整操作流程

内容准备 → 选择制作好的直播切片内容 → 商品关联 → 确定选中视频内容关联商品(系统自动识别/自行手动选择) → 发布设置 → 选择发布平台 → 同步发布指定的媒体

🚀 智能关联与发布系统

智能商品关联界面

视频内容关联商品选择界面，支持推荐关联、商品清单、商品库三种方式查找商品，提供精确的货号搜索功能，显示商品列表（包含商品ID、货号、AI推荐关联度），右上角显示已关联商品数量，便于快速完成商品关联配置

智能商品识别系统

系统通过AI智能识别技术，自动从视频内容中检测和识别商品特征，精准匹配商品库中的商品信息，实现商品与视频内容的智能关联，为后续的商品挂载和发布提供数据支持

多平台视频发布界面

立即发布功能页面，显示抖音和抖店两个主要发布平台选择，支持任务名称设置，提供添加视频功能（素材空间上传和本地上传两种方式），底部展示发布设置选项，包括挂车短视频和非挂车短视频两种发布模式选择

  • 系统自动识别商品

  • 精确搜索功能
  • 关联操作管理
